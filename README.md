# Aryan Agarwal – Web Developer Portfolio

A premium, modern single-page portfolio website showcasing web development expertise, full-stack architectures, and AI/ML integrations. 

🔗 **Live Website**: [portfolio-aryan8538.vercel.app](https://portfolio-9ci1z7yqf-aryan8538s-projects.vercel.app/)

---

## ✨ Features

- **Rich Aesthetics**: Premium glassmorphic design variables, gradient glows, and satisfying micro-animations.
- **Dark & Light Mode**: Seamless transitions with local storage caching to remember theme choices.
- **Scroll Progress Indicator**: Accent-colored progress indicator highlighting depth of read.
- **Responsive Media Queries**: Fully optimized experience on desktop, tablet, and mobile screens.
- **Live Demos**: Fully deployed, interactive web experiences for all showcased projects.
- **Web3Forms Contact Form**: Fully validated email submission system with zero server overhead.

---

## 🛠️ Technical Stack

- **Frontend**: Vanilla HTML5, Vanilla CSS3 (Custom Variables, Flexbox, CSS Grid), JavaScript (ES6+, IntersectionObserver APIs).
- **Backend integrations**: FastAPI, Node.js, REST APIs.
- **Libraries & Fonts**: Syne, Cabinet Grotesk, DM Mono (Google Fonts), Font Awesome 6.5.0.

## 🚀 Running Locally

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Aryan8538/portfolio.git
   cd portfolio
   ```

2. **Serve the directory** using any static HTTP server (e.g. Python):
   ```bash
   python -m http.server 8080
   ```

3. Open your browser and navigate to `http://localhost:8080`.
